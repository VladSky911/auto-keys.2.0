# AutoKluchi PRO – лендинг

Одностраничный лендинг для сервиса аварийного вскрытия и изготовления автоключей в Мюнхене.

- Чистый HTML/CSS/JS
- Мультиязычность: DE (по умолчанию), RU, EN через JSON‑файлы
- Реализовано:
  - плавный скролл по якорям
  - слайдер с типовыми ситуациями
  - модальное окно «обратный звонок»
  - формы с простым fake‑обработчиком (можно подключить backend)

Структура соответствует:

```text
autokluchi-pro/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
├── lang/
│   ├── de.json
│   ├── ru.json
│   └── en.json
└── assets/
    ├── icons/
    │   ├── key.svg
    │   ├── emergency.svg
    │   ├── chip.svg
    │   ├── repair.svg
    │   ├── location.svg
    │   └── clock.svg
    └── images/
        └── hero-bg.jpg
```
